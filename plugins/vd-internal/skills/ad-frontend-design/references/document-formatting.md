# Accelerate Data Document Formatting Standards

This reference provides complete specifications for formatting Excel, PowerPoint, and PDF documents according to Accelerate Data brand guidelines.

All values sourced from brand_book (AUTHORITATIVE SOURCE).

## Excel Spreadsheets

### Header Row (Row 1)
```
Background: Pacific (#00b4d8)
Font: Geist or system-ui, 14pt, Bold
Font Color: White (#ffffff)
Alignment: Center, Middle
Border: Thin, Pearl (#f2f2f2)
```

### Subheaders
```
Background: None
Font: Geist or system-ui, 14pt, Bold
Font Color: Navy (#03045e)
Alignment: Left, Middle
```

### Data Cells
```
Background: White or Pearl (alternating)
Font: Geist or system-ui, 14pt, Regular
Font Color: Navy (#03045e)
Alignment: Left for text, Right for numbers
Border: Thin, Pearl (#f2f2f2)
```

### Alternating Rows
```
Even rows: White (#ffffff)
Odd rows: Pearl (#f2f2f2)
```

### Chart Color Sequence
1. Pacific (#00b4d8) - Primary series
2. Seafoam (#00dd92) - Secondary series
3. Ocean (#0077b6) - Tertiary series
4. Navy (#03045e) - Quaternary series
5. Arctic (#90e0ef) - Fifth series
6. Powder (#caf0f8) - Sixth series

### Chart Formatting Rules
- Gridlines: Pearl (#f2f2f2), 0.5pt
- Axis labels: Navy (#03045e), 12pt
- Legend: Navy (#03045e), 12pt
- No 3D effects
- No gradients
- No decorative elements

## PowerPoint Presentations

### Slide Templates

#### 1. Title Slide
```
Background: Pacific (#00b4d8)
Title: White (#ffffff), 32pt, 600 weight
Subtitle: White (#ffffff), 18pt, 400 weight
Logo: Light variant, top-left, 1.5" width
Date/Presenter: White (#ffffff), 14pt, bottom area
```

#### 2. Section Divider
```
Background: Navy (#03045e)
Section Title: White (#ffffff), 32pt, 600 weight, centered
```

#### 3. Content Slide
```
Title Bar: Pacific (#00b4d8), 1" height
Title Text: White (#ffffff), 24pt, 600 weight
Content Background: Pearl (#f2f2f2) or White
Body Text: Navy (#03045e), 14pt, 400 weight
```

#### 4. Data Slide (Charts/Tables)
```
Background: White or Pearl
Title: Pacific (#00b4d8), 24pt, 600 weight
Charts: Use brand color sequence
Tables: Pacific headers, Pearl alternating rows
```

### Layout Rules
```
Margins: 0.5 inches all sides
Title position: Top 15% of slide
Content area: Top 20% to 90% of slide height
Bullet indentation: 0.25 inches per level
Maximum bullets: 6 per slide
```

### Bullet Styles
```
Level 1: Filled circle
Level 2: Open circle
Level 3: Filled square
Indent: 0.25 inches per level
```

### Footer
```
Height: 0.5 inches
Company name: Left aligned
Slide number: Right aligned
Optional date: Center aligned
Font: 11pt, Navy (#03045e)
```

## PDF Documents

### Page Layout
```
Page size: Letter (8.5" x 11")
Orientation: Portrait (default)
Margins: 1 inch all sides
Line spacing: 1.15
Paragraph spacing: 12pt after
```

### Header
```
Height: 0.75 inches
Left: Company logo (1.5" width)
Center: Document title, Navy (#03045e), 14pt
Right: Page X of Y, Navy (#03045e), 12pt
```

### Footer
```
Height: 0.5 inches
Left: Copyright notice, Smoke (#171c21), 11pt
Center: Date (Month DD, YYYY format)
Right: Classification (Confidential, etc.)
```

### Heading Styles

#### Heading 1 (Main Sections)
```
Font: Geist or SF Pro, 24pt, 600 weight
Color: Pacific (#00b4d8)
Spacing: 12pt after
Letter-spacing: -0.02em
```

#### Heading 2 (Subsections)
```
Font: Geist or SF Pro, 18pt, 600 weight
Color: Navy (#03045e)
Spacing: 10pt after
Letter-spacing: -0.02em
```

#### Heading 3 (Sub-subsections)
```
Font: Geist or SF Pro, 16pt, 600 weight
Color: Navy (#03045e)
Spacing: 8pt after
Letter-spacing: -0.015em
```

### Body Text
```
Font: Geist or SF Pro, 14pt, 400 weight
Color: Navy (#03045e)
Line spacing: 1.15
Paragraph spacing: 12pt after
```

### Captions
```
Font: Geist or SF Pro, 12pt, 500 weight
Color: Smoke (#171c21)
Style: Regular or italic for figure captions
```

### Table Formatting
```
Header row: Pacific (#00b4d8) background, White text, Bold
Data rows: Alternating White and Pearl backgrounds
Borders: Pearl (#f2f2f2), thin
Text: Navy (#03045e), 14pt
Numbers: Right-aligned
Text: Left-aligned
```

## Number Formatting Standards

### Currency
```
Format: $X,XXX.XX
Examples:
  $1,234.56
  $12,345,678.90
  ($1,234.56) for negative values
```

### Percentages
```
Format: XX.X%
Examples:
  45.5%
  100.0%
  -12.3%
```

### Large Numbers
```
Thousands: X.XK (e.g., 1.5K)
Millions: X.XM (e.g., 2.3M)
Billions: X.XB (e.g., 1.2B)
```

### Dates
```
Standard: Month DD, YYYY (January 15, 2026)
Short: MM/DD/YYYY (01/15/2026)
ISO: YYYY-MM-DD (2026-01-15)
```

## Accessibility Requirements

### Color Contrast
```
Body text on white: Navy (#03045e) - Ratio: 13.5:1
Body text on Pearl: Navy (#03045e) - Ratio: 12.8:1
White text on Pacific: White (#ffffff) - Ratio: 3.1:1
White text on Navy: White (#ffffff) - Ratio: 13.5:1
```

### Minimum Font Sizes
```
Print: 11pt minimum
Digital: 14px minimum
Captions: 12pt print / 12px digital
```

### Additional Requirements
- Never rely on color alone to convey meaning
- Provide alt text for all images and charts
- Use sufficient heading hierarchy
- Ensure tables have proper header row designation

## File Naming Conventions

### Standard Format
```
YYYY-MM-DD_DocumentType_Description_vX_Status.ext
```

### Examples
```
2026-01-15_Report_Q4-Revenue_v2_FINAL.xlsx
2026-01-15_Presentation_BoardMeeting_v1_DRAFT.pptx
2026-01-15_Document_PolicyUpdate_v3_APPROVED.pdf
```

### Status Values
- DRAFT - Work in progress
- REVIEW - Ready for review
- FINAL - Approved for distribution
- APPROVED - Formally approved
- ARCHIVED - No longer active

## Common Mistakes Checklist

Before finalizing any document, verify:

### Colors
- [ ] No colors outside the AD palette
- [ ] Pacific (#00b4d8) used for primary headers/CTAs
- [ ] Navy (#03045e) used for body text
- [ ] Seafoam (#00dd92) used sparingly for accents only
- [ ] No generic blues, greens, or other off-brand colors

### Typography
- [ ] Using Geist, SF Pro, or system fallbacks
- [ ] No decorative or script fonts
- [ ] Correct font weights (400 body, 600 headings)
- [ ] No Comic Sans, Papyrus, or similar fonts

### Layout
- [ ] Consistent margins throughout
- [ ] Logo properly placed and sized
- [ ] Proper heading hierarchy
- [ ] Adequate whitespace

### Data Presentation
- [ ] Numbers use comma separators
- [ ] Currency formatted as $X,XXX.XX
- [ ] Percentages show one decimal place
- [ ] Dates in Month DD, YYYY format
- [ ] Tables have alternating row colors

### Logo Usage
- [ ] Light logo variant on dark backgrounds
- [ ] Dark logo variant on light backgrounds
- [ ] Logo not stretched or distorted
- [ ] Adequate clear space around logo
